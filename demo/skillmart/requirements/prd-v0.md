# SkillMart PRD v0

SkillMart 是一个本地电商订单演示系统，用户可以查看商品、使用优惠券、创建订单、支付订单、查询订单和取消待支付订单。

当前需求只说明业务目标，尚未明确以下规则，因此测试侧不能直接进入完整测试用例（Test Cases）生成：

- 普通用户能否查询、支付、取消他人订单；
- 优惠券满减门槛、有效期和边界时间口径；
- 下单时库存是创建订单时锁定，还是支付时扣减；
- 相同幂等键重复提交时是否只能产生一张订单；
- 支付回调重复到达时是否允许重复改变业务状态；
- 订单状态有哪些，哪些状态允许取消；
- 页面、接口和数据验收口径是否必须一致；
- 错误码、鉴权、清理入口和测试数据恢复方式。

预期 `requirement-clarification-test` 输出 `P0 - BLOCKING`，并给出产品核对问题，不生成正式测试用例（Test Cases）。
