# testing-skills v1.0.0

首个标准安装包版本：七个测试工程 Skill 可被 Codex、Claude Code 和 CC Switch 发现与管理。

## Highlights

- 七个独立 Skill 名称保持不变，根目录中文文件作为唯一正文真源。
- 一个主 Skill、最多一个辅助 Skill；辅助调用前必须说明分工并等待用户确认。
- 五个正式用例生成 Skill 使用同一份数据同时输出 `.xlsx` 和离线 `.html`。
- Excel/WPS 支持四状态下拉、冻结表头、筛选和条件行颜色。
- HTML 支持搜索、模块/优先级/状态筛选、四状态统计和本地自动保存。
- `待定`表示已执行但多方口径存在歧义，暂时不能定性为缺陷。
- 完整版使用按需 references 降低上下文负担，不删除原门禁和表达。
- 提供 Codex 高质量 Excel 路径和无第三方依赖的 OOXML 兜底。

## Install all

```bash
npx skills add Saitamasans/testing-skills
```

逐个安装命令见 README。
